# Задатак 1

Написати програм на програмском језику Јава, који омогућава евиденцију,
обраду и анализу података о производима различитих типова
уз примену напредних концепата генеричких типова.

У пакету **artikli** постоји **Производ** који има::
* **Назив** и **цену**.
* Сви атрибути се задају приликом стварања.
* Могу се дохватити сви атрибути
* Приказује се у формату:
  ```
  naziv [cena]
  ```
  Пример:
  ```
  Hleb [90.0]
  ```

У пакету **artikli** постоји **Технички производ**.
**Технички производ** је **Производ** који има:
* поље гаранција (у месецима).
* Приказује се у формату:
  ```
  naziv [cena, garancija meseci]
  ```
  Пример:
  ```
  Televizor [48000.0, 24 meseci]
  ```

У пакету **prodavnica** постоји **Кутија** која има:
* Генерички параметар типа `T`
* Јединствени аутоматски генерисани идентификатор 
* Поље типа `T` које представља садржај кутије
* Генерички конструктор који прихвата садржај било ког типа и креира кутију:
  ```java
  public <U extends T> Kutija(U sadrzaj) { ... }
  ```
* Може се дохватити садржај
* Приказује се у формату:
  ```
  id. sadrzaj
  ```
  Пример:
  ```
  1. Hleb [90.0]
  ```

У пакету **prodavnica** постоји **Магацин** који има:
* Генерички параметар типа `T`
* Листу кутија типа Kutija<T>`
* Методу `dodaj(Kutija<? extends T> kutija)` која додаје кутију у магацин
(коришћење ограниченог џокера `extends`)
* Приказује се у формату:
  ```
  Magacin:
  id. sadrzaj
  ...
  ```
  Пример:
  ```
  Magacin:
  1. Hleb [90.0]
  2. Televizor [48000.0, 24 meseci]
  ```

У пакету **prodavnica** постоји интерфејс Analizator<T> који дефинише методу:
```java
double izracunaj(T vrednost);
```

Свака класа која имплементира овај интерфејс дефинише начин на
који се врши анализа података.

У пакету **analiza** постоји **Ценовни анализатор**.
**Ценовни анализатор** је **Анализатор** за **Производе**.
* Метода **izracunaj**, као параметар прима **Производ**,
  а враћа цену производа као резултат
* Омогућава израчунавање просечне цене свих производа у магацину.

Класа Statistika
У пакету **analiza** постоји **Статистика** која садржи:
* генеричку методу:
  ```java
  public static <T> double prosecnaVrednost(
     List<? extends T> lista, Analizator<? super T> analizator
  )
  ```
  која израчунава просек вредности добијених применом
  анализатора на сваком елементу листе.
  (Пример примене ? extends и ? super џокера.)

У пакету **prodavnica**, увести **Стари магацин** који користи **List<Object>** за чување производа.

У пакету **prodavnica**, постоји **MagacinAdapter** који уводи генерички тип и креира адаптер који омогућава
конверзију из **StariMagacin** у **Magacin<T>**:
```
public static <T> Magacin<T> konvertuj(StariMagacin stari)
```

У главном програму:
* Креирати неколико објеката **Производа** и **Техничких производа**
* Додати их у објекат **Magacin<Proizvod>**.
* Исписати садржај магацина.
* Припремити **Ценовни анализатор** за израчунавање просечне цена производа у магацину.
* Креирати **Стари магацин**, додати неке **производе**, а затим га конвертовати у **Magacin<Proizvod>**
помоћу **MagacinAdapter**-a.
* Исписати резултате пре и после конверзије и анализе.
