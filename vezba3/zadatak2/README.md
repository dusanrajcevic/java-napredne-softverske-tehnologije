# Задатак 2
Написати програм на програмском језику Јава, који води евиденцију о
различитим врстама банкарских рачуна уз коришћење **генеричких класа**.

У пакету **banka** постоји **Рачун** који има:
* Генерички параметар типа T (тип валуте)
* Јединствени **број рачуна** (аутоматски генерисан)
* Поље типа T које представља износ на рачуну
* Рачун се ствара са задатим износом
* Може се дохватити износ
* Приказује се у формату:
  ```
  broj_racuna. iznos
  ```
  *Пример*:
  ```
  1. 2500.0
  2. 1500.50
  ```

У пакету **banka** постоји класа **Клијент**. Клијент има:
* Име и презиме клијента
* Листу рачуна (типа **Racun<?>**)
* Приликом стварања задаје се име и презиме, а листа је празна
* Клијент може да дода нови рачун
* Приказује се у формату:
  ```
  Klijent: име презиме
  --- Racuni:
  број. износ
  ...
  ```
  *Пример*:Klijent: Marko Markovic
  ```
  --- Racuni:
  1. 2500.0
  2. 1500.50
  ```
У пакету **banka** постоји **Банка** која има:
* Генерички параметар типа T
* **Листу клијената** који имају рачуне са вредностима типа T
* Приликом стварања листа је празна
* Могуће је додати новог клијента
* Приказује се у формату:
  ```
  Banka:
  Klijent: ...
  ...
  ```
  *Пример*:
  ```
    Banka:
    Klijent: Marko Markovic
    --- Racuni:
    1. 2500.0
    2. 1500.50
    Klijent: Jelena Jovanovic
    --- Racuni:
    1. 3000.0
  ```

* Ограничени типови
  * Увести класу **FinansijskaBanka<T extends Number>** која наслеђује **Banka<T>** и омогућава
    израчунавање укупне суме свих рачуна свих клијената у банци. 
  * Приказује се у формату:
    ```
    Finansijska banka (ukupno=...)
    Klijent: ...
    ...
    ```
    *Пример*:
    ```
    Finansijska banka (ukupno=7000.5)
    Klijent: Marko Markovic
    --- Racuni:
    1. 2500.0
    2. 1500.50
    Klijent: Jelena Jovanovic
    --- Racuni:
    1. 3000.0
    ```

* Џокери (wildcards)
  * Увести статичку методу `public static <T> void prebaciKlijente(Banka<? extends T> izvor, Banka<? super T> odrediste)`
    која пребацује све клијенте из једне банке у другу:
    * izvor производи клијенте (користи extends)
    * odrediste конзумира клијенте (користи super)
  * Након пребацивања, изворна банка остаје празна.

У главном програму:
* Креирати две банке:
  * Банку "Nacionalna Banka" за тип Double
  * Банку "Univerzalna Banka" за тип Object
* Креирати три клијента (са предефинисаним именима) и додати им рачуне са износима различитих типова (Double, Integer, итд.)
* Приказати банке пре и након пребацивања клијената.
* Креирати **FinansijskuBanku** и израчунати укупну суму новца у свим рачунима.
* Демонстрирати употребу методе prebaciKlijente тако што ће се клијенти пребацити из Banka<Double> у Banka<Object>.
