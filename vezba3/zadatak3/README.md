# Задатак 3

Написати програм на програмском језику Јава, који води евиденцију
о пошиљкама у курирској служби уз коришћење **генеричких класа**.

У пакету **postanska_sluzba** постоји **Пошиљка** која има:
* Генерички параметар типа T (тип садржаја пошиљке)
* Јединствени аутоматски генерисани **идентификатор** пошиљке
* Поље типа T које представља **садржај пошиљке**
* Поље **маса** (у килограмима)
* Пошиљка се ствара са задатим садржајем и масом
* Може се дохватити садржај и маса
* Приказује се у формату:
  ```
  id. sadrzaj [masa kg]
  ```
  *Пример*:
  ```
  1. Knjiga: Na Drini cuprija [2.5 kg]
  2. Telefon: Samsung S24 [0.8 kg]
  ```

У пакету **postanska_sluzba** постоји Skladiste које има:
* Генерички параметар типа T
* **Листу пошиљака** типа Posiljka<T>
  * Листа се креира празна
* Могуће је додати нову пошиљку у складиште
* Приказује се у формату:
  ```
  Skladiste:
  id. sadrzaj [tezina kg]
  ...
  ```
  *Пример*:
  ```
  Skladiste:
  1. Knjiga: Na Drini cuprija [2.5 kg]
  2. Telefon: Samsung S24 [0.8 kg]
  ```

Ограничени типови:
* Увести класу **TezinskoSkladiste<T extends Number>** која наслеђује **Skladiste<T>**
  и омогућава израчунавање укупне масе свих пошиљака у складишту.
* Приказује се у формату:
  ```
  Tezinsko skladiste (ukupna tezina=...)
  id. sadrzaj [tezina kg]
  ...
  ```
  *Пример*:
  ```
  Tezinsko skladiste (ukupna tezina=3.3)
  1. Knjiga: Na Drini cuprija [2.5 kg]
  2. Telefon: Samsung S24 [0.8 kg]
  ```

Џокери (wildcards):
* Увести статичку методу:
  `public static <T> void prebaciPosiljke(Skladiste<? extends T> izvor, Skladiste<? super T> odrediste)`
  која пребацује све пошиљке из једног складишта у друго:
  * izvor производи елементе (користи extends)
  * odrediste конзумира елементе (користи super)
* Након пребацивања, изворно складиште остаје празно.

Главни програм
* Креирати складиште књига и додати неколико пошиљака са садржајем типа `Knjiga`.
* Креирати складиште техничке робе (нпр. Telefon, Laptop) и додати неколико пошиљака.
* Приказати оба складишта.
* Креирати `TezinskoSkladiste` које садржи пошиљке са бројевима (`Double`, `Integer`) који представљају тежине.
* Приказати укупну тежину пошиљака у том складишту.
* Демонстрирати употребу методе `prebaciPosiljke` тако што ће се пошиљке пребацити из `Skladiste<Knjiga>` у `Skladiste<Object>`.
