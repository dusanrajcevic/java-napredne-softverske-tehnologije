# Задатак 1

Написати програм на програмском језику Јава, који води евиденцију о колекцијама елемената различитих типова уз коришћење **генеричких класа**.


У пакету **artikli** постоје:
* **Књига** која има: назив и аутора
    * Приказује се у формату:
      ```
      Knjiga: naziv - autor
      ```
      *Пример*:
      ```
      Knjiga: Na Drini cuprija - Ivo Andric
      ```
* **Телефон** који има: модел и цену
    * Приказује се у формату:
      ```
      Telefon: model [cena]
      ```
      *Пример*:
      ```
      Telefon: Samsung S24 [1200.0]
      ```

У пакету **kolekcija** постоји **Кутија** која има:
* Генерички параметар типа **T**
* Јединствени аутоматски генерисани идентификатор
* Поље типа *T* које представља **садржај** кутије
* Кутија се ствара са задатим **садржајем**
* Може се дохватити **садржај** кутије
* Приказује се у формату:
  ```
  id. sadrzaj
  ```
  *Пример*:
  ```
  1. Knjiga: Na Drini cuprija
  2. Broj: 42
  ```

У пакету **kolekcija** постоји **Магацин** који има:
* Генерички параметар типа **T**
* Листу кутија типа **Kutija<T>**
* Листа кутија се ствара празна
* Постоји метода која омогућава додавање **кутије** у магацин
* Приказује се у формату:
  ```
  Magacin:
      id. sadrzaj
      ...
  ```

* Ограничени типови:
  * Увести класу **NumerickiMagacin<T extends Number>** која наслеђује **Magacin<T>** и омогућава
    рачунање збирa свих вредности у кутијама.
  * Приказује се у формату:
    ```
    Numericki magacin (zbir=...)
      id. sadrzaj
      ...
    ```
* Џокери (wildcards):
  * Увести методу `public static <T> void prebaciSadrzaj(Magacin<? extends T> izvor, Magacin<? extends T> odrediste)`
    која пребацује све кутије из једног магацина у други:
    * izvor производи елементе (користи се extends)
    * odrediste конзумира елементе (користи се extends)
  * Након пребацивања, изворни магацин остаје празан.

У главном програму:
* Креирати магацин књига и додати у њега две кутије са књигама.
* Креирати магацин телефона и додати у њега два телефона.
* Приказати оба магацина.
* Креирати нумерички магацин и додати у њега неколико бројева (нпр. целих и реалних).
* Приказати нумерички магацин и израчунати збир.
* Демонстрирати употребу методе `prebaciSadrzaj` тако што ће се пребацити све кутије из једног магацина у други (нпр. из `Magacin<Knjiga>` у `Magacin<Object>`).
